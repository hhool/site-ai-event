# Epic PRD：年度现象级开源 AI 工具集网站

> English version: [epic.md](epic.md)

## 1. Epic 名称

**年度现象级开源 AI 工具集**

---

## 2. 目标

### 问题

随着 AI 领域高速发展，每年涌现出大量开源 AI 工具，但缺乏一个系统性、权威性的年度盘点平台。用户难以快速了解某年度最具代表性的开源 AI 工具，也缺乏深度、系列化的产品级介绍内容。

### 解决方案

构建一个以"年份"为轴的静态网站，收录 2023–2025 年各年度 ≤10 款现象级开源 AI 工具，每款工具提供系列化深度介绍（背景 / 技术亮点 / 社区反响 / 适用人群），并附官方 Demo 入口、GitHub 链接、分类标签，支持中英双语切换。

### 预期价值

- 成为 AI 从业者和爱好者的年度工具索引标杆内容
- 通过高质量 SEO 内容持续获取搜索流量
- 双语内容覆盖中英文受众，扩大传播范围

---

## 3. 用户画像

1. **AI 从业者** — 工程师、研究员，需要快速了解某年度开源 AI 生态全貌
2. **产品经理 / 创业者** — 寻找可集成或参考的开源 AI 工具
3. **AI 爱好者 / 学习者** — 希望了解现象级工具的技术背景和使用方式
4. **内容创作者 / 媒体** — 用作年度 AI 盘点内容的参考来源

---

## 4. 核心用户旅程

**旅程 1：年度工具浏览**
> 用户进入主页 → 选择年份（2023/2024/2025）→ 浏览该年工具卡片网格 → 按分类筛选 → 点击感兴趣的工具卡片

**旅程 2：工具深度了解**
> 用户点击工具卡片 → 进入详情页 → 依次阅读「背景 → 技术亮点 → 社区反响 → 适用人群」→ 点击"立即体验"进入官方 Demo

**旅程 3：跨语言使用**
> 用户点击右上角 ZH/EN 切换 → 所有 UI 文案与工具介绍内容同步切换为目标语言

---

## 5. 业务需求

### 功能需求

- 主页支持按年份（2023 / 2024 / 2025）筛选工具
- 主页支持按分类标签多选筛选
- 每个工具有独立详情页 `/[locale]/tool/[slug]`
- 详情页包含四节系列化内容：背景、技术亮点（3–5条）、社区反响、适用人群（3–4类）
- 每个工具必须有有效的官方 Demo 入口链接（`demoUrl` 必填字段）
- 全站支持中英双语（URL 策略：`/zh/` 和 `/en/`）
- 右上角提供 ZH/EN 语言切换按钮
- 所有工具必须为开源项目（有 GitHub 链接）
- 工具数量：每年 ≤10 款，共 3 年

### 非功能需求

- **性能**：Lighthouse Performance ≥ 85，首屏加载 < 2s
- **SEO**：每页独立 `<title>` / `<meta description>` / `hreflang`，Lighthouse SEO ≥ 90
- **可访问性**：Lighthouse Accessibility ≥ 85
- **部署**：Vercel，构建产物为静态文件（SSG），零运行时成本
- **可维护性**：新增年份工具只需在 `tools.ts` 追加数据，无需修改组件
- **安全**：所有外部链接使用 `rel="noopener noreferrer"`
